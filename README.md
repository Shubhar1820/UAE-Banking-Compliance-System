# 🏦 UAE Banking Compliance System

## Project Overview
Automated AML/KYC Compliance ETL Pipeline 
for UAE Banking — built using Python & SQLite.

## Features
- ✅ AML Transaction Monitoring
- ✅ KYC Verification System
- ✅ Real Time Fraud Detection (5 patterns)
- ✅ Risk Scoring Engine
- ✅ SQLite Database with SQL Views
- ✅ 6 Chart Compliance Dashboard
- ✅ Automated Compliance Report

## Tools Used
- Python
- Pandas
- SQLite
- Matplotlib

## Dataset
- 10,000 realistic UAE banking transactions
- 500 customers
- 10 UAE branches
- 7 currencies

## Results
- 🔴 1,016 AML flagged transactions
- ⚠️ 1,497 KYC failures detected
- 🚨 1,036 fraud cases identified
- 🔴 93 CRITICAL risk transactions

## Industry
Banking | FinTech | Insurance | UAE Market
